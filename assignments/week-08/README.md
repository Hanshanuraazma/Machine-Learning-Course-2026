# 🏆 Ujian Tengah Semester (UTS): End-to-End Machine Learning Challenge

## Deskripsi Tugas
Selamat datang di tantangan tengah semester! Tugas Anda adalah membangun pipeline Machine Learning lengkap, mulai dari eksplorasi data hingga evaluasi model tingkat lanjut menggunakan teknologi yang telah dipelajari (GitHub & Weights & Biases).

## Pilihan Kasus (Pilih Salah Satu)
1. **Financial Risk (Classification):** Deteksi kelayakan kredit atau penipuan (dataset imbalanced).
2. **Medical Diagnosis (Classification):** Prediksi penyakit kronis (Diabetes/Jantung).
3. **Consumer Behavior (Regression):** Prediksi harga properti atau estimasi permintaan barang.

## Persyaratan Teknis
1. **Version Control:** Kode wajib diunggah ke GitHub melalui mekanisme Pull Request (PR).
2. **Experiment Tracking:** Gunakan **Weights & Biases** untuk mencatat minimal 3 eksperimen model yang berbeda (misal: Logistic Regression vs Random Forest vs XGBoost).
3. **Imbalance Handling:** Jika menggunakan data klasifikasi yang timpang, wajib menggunakan teknik SMOTE atau penyesuaian bobot kelas.
4. **Deep Analysis:** Wajib menyertakan analisis Confusion Matrix dan Feature Importance.

| Kriteria | Bobot | Indikator Keberhasilan |
